# Rewire: Privacy-First Digital Wellness

## Product Scope (MVP)
The MVP focuses on the core "Silent Engine" and the "Calm Friend" UI.
- One-action-at-a-time home screen.
- Emergency "I need a minute" mode.
- 5 Silent Engines for progress tracking.
- Local-only data storage (Dexie).

## Sitemap & Routes
- `/`: Home (Decision engine picks the ONE action)
- `/checkin`: Daily conversational check-in
- `/panic`: Emergency breathing and grounding
- `/reflect`: Weekly narrative reflection
- `/settings`: Privacy and encryption settings

## UI Spec
- **Tone**: Warm, non-clinical, supportive.
- **Layout**: Centered content, soft edges, organic colors.
- **Micro-copy**: "Take a breath," "You've got this," "It's a process."

## Phased Build Plan
1. Core Engines (Logic modules)
2. Database layer (Dexie)
3. State management (Zustand)
4. Layout & Design System (Shadcn + Tailwind)
5. Home Decisions Engine
6. Check-in Interface
7. Reflection & Narrative Generator
8. Ethical Safeguards & Final Polish
